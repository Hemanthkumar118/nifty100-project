# Nifty100 Financial Intelligence Platform

## Overview

A Financial Intelligence Platform developed using Python, SQLite, Pandas, Plotly and Streamlit.

---

## Features

- ETL Pipeline
- Financial Ratio Engine
- Stock Screener
- Peer Comparison
- Valuation Module
- Dashboard
- Trend Analysis
- Reports

---

## Tech Stack

- Python
- Pandas
- SQLite
- Streamlit
- Plotly
- OpenPyXL

---

## Project Structure

- Sprint 1 – Data Foundation
- Sprint 2 – Financial Ratio Engine
- Sprint 3 – Stock Screener
- Sprint 4 – Dashboard & Analytics
